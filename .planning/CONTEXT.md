## Functional Requirements

- Build and maintain backend services in Golang for a B2B SaaS platform (Grux product)
- Develop and maintain cross-platform mobile/web frontend applications using Flutter (Dart) with JavaScript interoperability
- Implement cloud-native backend services on Google Cloud Platform (GCP) including deployment, monitoring, and operations
- Cross-train across two distinct products (Grux + Rook) to provide coverage and knowledge sharing across the product suite
- Integrate with Google APIs and GCP services (e.g., Cloud Storage, Cloud Functions, Pub/Sub, or equivalent managed services)
- Store and retrieve data using cloud-native patterns (microservices, RESTful APIs, or gRPC)
- Validate and export data as required by B2B SaaS business workflows

## Non-Functional Requirements

- **Production-Grade**: Code must meet production readiness standards (testing, error handling, logging, observability)
- **Scalability**: Cloud-native architecture must support scaling for B2B SaaS workloads
- **Availability**: Services must be deployable with high availability on GCP managed infrastructure
- **Performance**: Responsive mobile/web frontend with <3s initial load on typical connections
- **Security**: Secure coding practices for B2B data; no PHI/PII requirements specified but enterprise data protection implied
- **Reliability**: Logging, monitoring, and error tracking for production services
- **Maintainability**: Clean architecture enabling cross-training across two products

## Constraints

- **Tech Stack**: Golang (Go) for backend; Flutter/Dart with JavaScript for frontend; GCP for cloud infrastructure (Node.js acceptable as alternative, Chrome Extensions tangential)
- **Timeline**: Contract duration >6 months; full-time commitment >30 hours/week
- **Availability**: Must overlap Pacific Time (US West Coast) during 8:00 AM - 5:00 PM PT
- **Location**: Must be in approved regions/time zones (US-based company with global team)
- **Employment Type**: Contract-to-hire model
- **Dual Product Coverage**: Must train across both Grux and Rook products
- **Language**: Fluent English required

## Success Criteria

- Deliver functioning backend API endpoints in Golang deployed to GCP
- Produce working Flutter frontend features with Dart code compiling and running
- Pass code review demonstrating Go + Flutter production patterns (test coverage, error handling)
- Complete cross-training on Rook product within first 4 weeks
- Maintain >30 hrs/week availability with consistent PT overlap throughout contract

## Out of Scope (Initial)

- Native mobile development outside Flutter (iOS Swift, Android Kotlin)
- Chrome Extension development (listed as detectable but not core requirement)
- Full-stack with Node.js as primary backend (acceptable alternative only if needed)
- Building new products outside Grux and Rook
- DevOps/infrastructure beyond application-level GCP deployment (managed by platform team)
